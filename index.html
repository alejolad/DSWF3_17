HTML5

HTML significa Hyper Text Markup Language. HTML es la base de todas las cosas de la Web y es una habilidad necesaria para cualquier desarrollador web. Casi todos los sitios web se compone de HTML si es una variante de HTML.

CSS3

CSS significa Cascading Style Sheets y proporciona HTML con diseño y el diseño. Además de hacer cosas bastante y estéticamente agradable, CSS también proporciona una estructura general de HTML

JQUERY

Es una libreria que empaqueta un conjunto de sentencias listas para ser usadas por el diseñador de páginas, y estas permiten que el diseño sea lo más profesional posible. Por otra parte, una de las ventajas más grandes que presenta jQuery es la compaƟbilidad de su código con todos los navegadores, lo que permite al diseñador despreocuparse de las conﬁguraciones correspondientes.

JAVASCRIPT

JavaScript es un lenguaje de programación dinámico utilizado para hacer páginas web interactivas. Su intérprete está incrustado en el software del navegador web, como Internet Explorer, Chrome y Firefox, etc. Se ejecuta en el ordenador y el navegador del visitante.

BOOTSTRAP

Es framework (estructura basica de codigo) es uno de los más populares del mercado, habiendo sido desarrollado por el equipo de Twitter. Bootstrap ha sido creado pensando en ofrecer la mejor experiencia de usuario tanto a usuarios de PC (IE7
incluido!), como a smartphones y tabletas.
