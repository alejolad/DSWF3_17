INTRODUCCIÓN

Este ovi es creado con el fin de dar a conocer conceptos, características y rutinas necesarios para la creación de sitios WEB, en su contenido se van a encontrar temas relacionados directamente con HTML, HTML5, CCS3 y temas relacionados directamente con estos conceptos como lo son javascript, bootstrap, php, todos ellos etan integrados directamente con la programación y la creación de sitios WEB.

OBJETIVOS

Objetivo general:
• Identificar los requerimientos para la construcción de Objetos Virtuales Informativos básicos utilizando lenguaje de marca HTML5 y CSS.
Objetivos específicos
• Conocer y seleccionar una de las herramientas de control de versiones y editor de código para el inicio de un proyecto web.
• Identificar las etiquetas básicas del lenguaje de marcado HTML5 para la maquetación básica de un Objeto Virtual Informativo.
• Identificar los elementos básicos de aplicación de estilos, utilizando CSS3 para un Objeto Virtual Informativo.
